What will you do with your data when you die?

D-Public

Put my data in the public domain

Remix my data freely with that of anyone else

Recreate version of my former self algorithmically

NB: this license can be augmented by an NC or C amendment to restrict commercial use or make into a free data license.

D-Erase

Erase my data from all public and private sources

Send cease and desist letters to any source that continues to host my data

Counter data remanence by all possible means, including reformatting or the physical destruction of storage media

D-Science

Donate my data to science and grant access to accredited researchers only

Attribute posthumously, any insight derived from my data

NB: this license can be augmented by an NC or C amendment to restrict commercial use or make into a free data license.

View legal code »

D-Friends/Family

Allow friends and family to impersonate me online.

Grant login credentials to the people specified in Appendix C of the legal code for all my online accounts.

View legal code »

Data-Fake

Rewrite my data such that my former self is no longer recognizable

Send my data through a scrambler, pollute it as best you can

Continue to create data (since it will anyhow be impossible to erase any of it)
